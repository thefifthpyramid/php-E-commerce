# Database details

::: note:
*dataBase Name : 'php_e-commerce'*
:::

## Database tables

1. users
   + id
   + userName
   + password
   + email
   + fullName
   + groupID 'Valeur par défaut = 0'

2. categories
    + id
    + name
    + description
    + sort
    + visibility 'Valeur par défaut = 0'
    + allow_comment 'Valeur par défaut = 0'
    + allow_ads 'Valeur par défaut = 0'
